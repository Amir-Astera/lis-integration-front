# LIMS Dashboard Frontend

Фронтенд-прототип для информационно-аналитической системы лаборатории государственной больницы.

## Что реализовано

- React + Vite приложение
- Главный экран `Сводка лаборатории`
- Визуальный перенос базового утвержденного макета в браузерный интерфейс
- Структура данных подготовлена для последующей интеграции с REST API

## Как запустить

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
